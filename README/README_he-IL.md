## 📖 מבוא

NuMonitor הוא כלי ניטור יציאה טורית המיועד למפתחי **Arduino/ESP32**, שמטרתו להחליף את Serial Monitor של Arduino IDE עם תכונות משופרות.

### ✨ תכונות עיקריות

- 🖥️ **צג טרמינל** - הצגת נתונים טוריים בזמן אמת
- 📊 **7 סוגי תרשימים** - קו, שטח, עמודות, פיזור, עוגה, מד, שטח מוערם
- 🗺️ **מעקב GPS** - אינטגרציה עם OpenStreetMap למסלולי GPS
- 🎯 **סמן ציר X** - ריחוף עכבר מציג ערכי נקודות נתונים מיידית
- 🌐 **18 שפות** - זיהוי אוטומטי של שפת הדפדפן
- 📦 **קובץ יחיד** - ללא צורך בהתקנה
